Setup to use the App

To use this webapp you need to setup our custom webserver. The source files can be found on our github under the folder httpServer. The server is written in go and can be compiled with a normal go-compiler( go for windows, mac and linux, gcc-go with binaries for linux)

  1. You need to get a copy of our source code. You can clone clone the repo with git clone or just download the zip file. We won't be realesing any binaries, so you need to compile it youself.
  2. Next you need to compile the server. If you decided to download the standart go compiler you just need to run go build inside the desired directory and the program should be compiled after a while. For the gcc option you will need to dig into the documentation. We may make a makefile for easier use in the future
  3. The rest should be straight forward, you just need to run ./httpServer <port> (.\ for windows). If you want to run this program from anywhere (for whatever reason) you could put it inside you PATH variable. But we don't recommend this.

Further Reading

We don't recommend to run this server on a public lan or in any lan where you don't trust everybody. The service is also not meant as an secure messaging plattform. It should be used for educational purposes only. You shoudn't send any sensitive data over it.
